### What I do

I design infrastructure that makes AI agents reliable — persistent memory, structured execution, and automated verification so agents **plan and execute** instead of guessing.

> *"Documentation isn't about the code — it instructs who will build the next step."*

---

### How I work

```
 RESEARCH   →  Deep technical research & trade-off analysis
 PLAN       →  Executable plans with YAML metadata & acceptance criteria
 EXECUTE    →  Code implementation based on defined criteria
 VERIFY     →  Automated validation against pre-defined truths & artifacts
```

The GSD cycle — TDD applied at the project level. The agent researches, plans, executes, and self-verifies against criteria defined **before** execution.
